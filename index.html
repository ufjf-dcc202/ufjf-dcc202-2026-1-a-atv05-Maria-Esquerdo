Desenvolvimento Web - Atividade 01

  • Objetivo:

    Por meio da atividade proposta pelo professor, o intuito desta página é colocar em prática alguns conhecimentos aprendidos acerca da linguagem de marcação HTML em aula, relembrar alguns conceitos já aprendidos anteriormente, de forma autônoma, e apresentar de forma organizada, estruturas básicas e essenciais da linguagem.

    Clique para mais informações!

    O HTML é uma linguagem de marcação e, puro, apresenta todo o conteúdo estruturado, mas sem estilos visuais ou comportamentos dinâmicos. O ideal é combinar HTML com CSS para estilização e JavaScript para interatividade, em conformidade com o planejamento da disciplina.

  • Desenvolvimento:

    O desenvolviemnto será estruturado como um breve e básico resumo explicativo de conceitos do HTML.

    1. Estrutura básica do documento HTML:

      • Declaração: !DOCTYPE html
      • Tags: html, head (com meta charset="UTF-8" e title) e body

    2. Elementos semânticos de estrutura:

      • header: cabeçalho da página.
      • nav: menu de navegação.
      • main: conteúdo principal.
      • section: divisão por temas.
      • article: conteúdo independnete.
      • aside: informações complementares.
      • footer: rodapé.

    3. Elementos de conteúdo:

      • Títulos hierárquicos. (h1 a h6)
      • Parágrafos (p).
      • Listas ordenadas (ol) e não ordenadas (ul).
      • Imagens(img com src, alt e title).
      • Links (a, com atributo href).
      • Tabelas (table, tr, th, td)
      • Formulários (form, input, label, button).

    4. Elementos de agrupamento:

      • div: para agrupamento genérico (estilização com definição de classes).
      • span: para elementos inline.

    5. Comentários em HTML:

      • A linguagem também permite a utilização de comentários no código, os quais não aparecem na página.

  • Conclusão:

    Este exercício permitiu consolidar na prática os conceitos de marcação HTML, reforçando a importância de uma estrutura semântica, bem organizada e acessível. O conhecimento adquirido serve como alicerce para as próximas etapas do desenvolvimento web.
